Descripcion

Microsoft SQL Server 2014

Microsoft SQL Server es un sistema de gestión de base de datos relacional, desarrollado por la empresa Microsoft. El lenguaje de desarrollo utilizado es Transact-SQL, una implementación del estándar ANSI del lenguaje SQL, utilizado para manipular y recuperar datos, crear tablas y definir relaciones entre ellas.

Con esta alternativa de entrenamiento de Base de Datos Microsoft SQL Server podrás darle mayor seguridad, estabilidad y performance a tus sistemas o proyectos.

Aprenderás:

  • Instalación de la Base de datos.
  • Instalación del IDE SQL Server Management Studio.
  • Creación de Base de Datos, Tablas, Columnas de información, Tipos de datos.
  • Insertar, Actualizar, Eliminar y Consultar registros.
  • Manejo y Uso de la Sentencia SQL.
  • Funciones Strings, Numéricas, Fechas y Avanzadas de SQL Server.
  • Creación de tablas temporales
  • Creaciones de Vistas.
  • Consultas Estructuradas con SQL
  • Creaciones de Procedimientos Almacenados - Nivel Básico.
  • Creaciones de Procedimientos Almacenados - Nivel Intermedio.
  • Creaciones de Procedimientos Almacenados - Nivel Avanzado.
