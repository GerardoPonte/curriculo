Active Directory - Directorio Activo: de cero a arquitecto

  • Videos: 356
  • Total, de horas: 48 horas.
  • Nivel de habilidad requerida: intermedio.
  • Recursos descargables: 57.

Descripcion

El objetivo principal del curso esta relacionado con la arquitectura detallada del Directorio Activo de Microsoft, tiene como finalidad llevar al asistente a una comprensión completa de toda la arquitectura corporativa y aplicable del Directorio Activo a la vida real empresarial.

Como claramente se determina en su titulo “Active Directory - Directorio Activo: de cero a arquitecto”, los conocimientos que debe tener el asistente como requisito para tomar este curso es CERO, es decir, no necesita conocimientos previos de Directorio Activo; paso a paso, la capacitación, nos llevara a un entendimiento profesional del servicio como tal.

No olvides que este servicio rey del sistema operativo Windows Server y esta completísima capacitación te hará brillar de manera espectacular ante los diferentes escenarios corporativos que se te presenten y ante las complejas solicitudes que te realicen los usuarios VIP ’s de las diferentes corporaciones en las cuales implementaras este servicio.

Esta capacitación te dará la habilidad de poder resolver temas tales como:

  • Implementación de diversos escenarios de Controladores de Dominio.
  • Diseños lógicos.
  • Diseños físicos.
  • Aplicación de políticas para usuarios.
  • Aplicación de políticas de seguridad.
  • Distribución de software.
  • Alta disponibilidad.
  • Recuperación de desastres.
  • Azure.
  • POWERSHELL.

Lo que aprenderás:

  • Conceptualización continua en Arquitectura de Active Directory
  • Implementación de DNS para Directorio Activo
  • Implementación de diferentes escenarios corporativos de Directorio Activo
  • Arquitectura de objetos de Directorio Activo
  • Arquitectura de GPO's
  • Seguridad tecnológica corporativa, aplicada desde el Directorio Activo
  • Arquitectura de sitios
  • Arquitectura de replicación
  • Alta disponibilidad de controladores de dominio
  • Recuperación de desastres de Directorio Activo
  • Distribución de software a la red corporativa
  • Arquitectura de operaciones maestras
  • Diseño de bosques, arboles, dominios y relaciones de confianza externas
  • Migración de dominios
  • Windows Server 2019
  • Windows Server 2022
  • PowerShell para Active Directory
  • Azure para Active Directory
